# Neutral Calibration Prompts

Use these prompts to calibrate style without reusing private topics from source examples. Categories are suggestions only; users may create arbitrary profile names.

## Email

- Write a brief follow-up asking whether a teammate has the information needed to move a task forward.
- Write a status update explaining that a small delay happened, what is still on track, and what happens next.
- Write a polite note declining an optional meeting while offering an alternate way to contribute.
- Rewrite a long scheduling note into a clearer message with the same facts.
- Write a warm thank-you note after someone reviewed a draft.

## Text

- Write a short message saying you are running ten minutes late and will follow up soon.
- Write a casual check-in asking whether plans for later still work.
- Rewrite a wordy text into something concise but not abrupt.
- Write a quick response that acknowledges good news without overdoing it.
- Write a message asking for a small favor with low pressure.

## Report

- Write a concise project summary with context, current state, risks, and next steps.
- Rewrite a dense paragraph into a clearer executive summary.
- Write a recommendation memo comparing two reasonable options.
- Write a neutral incident summary that separates observed facts from interpretation.
- Write a progress update for readers who need scanning-friendly structure.

## Narrative

- Write a short first-person scene about arriving somewhere unfamiliar.
- Rewrite a reflective paragraph so it feels more grounded and less abstract.
- Write a character observation without explaining the character directly.
- Write a transition from a quiet moment into a difficult conversation.
- Write a closing paragraph that resolves emotion without overexplaining.

## General Purpose

- Explain a complicated idea to a smart reader who is new to the topic.
- Rewrite a generic paragraph so it sounds more specific and human.
- Write a short invitation to collaborate on a small piece of work.
- Write a note that gives direct feedback while preserving trust.
- Summarize a decision, the reason for it, and the next action.

## Custom Prompt Handling

If the user provides a custom calibration prompt, treat it as a one-time prompt unless they explicitly confirm it should influence persistent profile guidance. Do not save private facts, names, internal details, or source-specific claims from a custom prompt into the profile by default.
